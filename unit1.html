Q2. Give short answers to the following questions.

i. What is a Computer?
A computer is an advanced electronic device that takes raw data as input from the user and processes it under the control of a set of instructions called a program. It produces the result as output and saves it for future use. A computer system consists of two main parts: hardware (physical components) and software (instructions). It can perform complex calculations and logical operations with high speed and accuracy, making it essential in modern life for communication, education, and business.

ii. What is processing operation?
Processing operation is the core function of a computer where the Central Processing Unit (CPU) executes instructions to transform raw data into meaningful information. During this stage, the computer performs arithmetic calculations (like addition or subtraction) and logical comparisons (like greater than or equal to). It is considered the most important step because it determines how the input data is manipulated to produce the desired results. All data movement and manipulation within the system happen during this operation.

iii. Show basic operations of a computer with the help of a block diagram.
The basic operations of a computer follow the IPOS cycle: 1. Input: Capturing data. 2. Processing: CPU manipulating data. 3. Output: Displaying results. 4. Storage: Saving data for later. (Note: In a block diagram, arrows flow from Input → CPU (Processing) → Output, while Storage is connected bi-directionally to the CPU).

iv. What is a notebook computer?
A notebook computer, commonly known as a laptop, is a small, portable personal computer designed for mobile use. It combines all the components of a desktop, including the screen, keyboard, pointing device (touchpad), and speakers, into a single battery-operated unit. Notebooks are powerful enough to run standard software and are preferred by students and professionals who need to work while traveling. Modern notebooks are extremely thin (ultrabooks) and offer high performance comparable to desktop systems.

v. State five differences between hardware and software.
1. Hardware is the physical parts of a computer you can touch, while software is a set of instructions you cannot touch. 2. Hardware is manufactured in factories, whereas software is developed by programmers. 3. Hardware cannot perform any task without software, and software cannot be executed without hardware. 4. Hardware can wear out over time, but software does not wear out but can become outdated. 5. If hardware is damaged, it is replaced; if software is damaged, it is reinstalled using a backup.

vi. Differentiate between an Interpreter and a compiler.
An Interpreter translates the high-level language program into machine code line-by-line. If it finds an error, it stops execution immediately. On the other hand, a Compiler translates the entire program at once into an object file. Compilers are generally faster for execution because the whole code is ready at once, while Interpreters are better for debugging as they show errors as they occur. Examples include Python (Interpreter) and C++ (Compiler).

vii. How Application software help Users?
Application software is designed to perform specific tasks for the user, rather than for the computer system itself. It helps users by automating manual work, such as writing letters in Word processors, managing accounts in Spreadsheets, or creating presentations. It also provides entertainment through games and media players. Essentially, application software turns the computer into a useful tool for various professional and personal needs, allowing users to be more productive and creative.

viii. Differentiate between shareware and freeware.
Freeware is software that is available at no cost for an unlimited period, like Google Chrome or Skype. Shareware, however, is "trialware" provided for free only for a limited time (e.g., 30 days). After the trial ends, the user must pay to continue using all features. Freeware is usually fully functional, while shareware often has limited features until the full version is purchased. Both are copyrighted and cannot be sold or modified by the user without permission.

ix. What is Licensed Software?
Licensed software is software that is legally protected and requires a "License Key" or "Product Key" to be used. When a user buys this software, they are not buying the code itself but the right to use it under specific conditions defined by the End User License Agreement (EULA). This prevents illegal copying and distribution. Examples include Microsoft Windows and Adobe Photoshop. Using such software without a valid license is considered piracy and is a legal offense.

x. What is firmware?
Firmware is a specific type of software that is permanently "embedded" or programmed into a hardware device's memory (usually ROM). It provides the necessary instructions on how the device communicates with other computer hardware. Unlike regular software, firmware is not designed to be frequently changed. It is found in devices like remote controls, digital watches, and the BIOS of a computer. It acts as the "glue" that allows hardware to start up and function correctly.

Q3. Give detailed answers to the following questions.

i. What are computing devices? Explain early and modern computing devices.

Computing devices are machines capable of receiving, processing, and outputting data. Evolution of these devices is divided into early and modern eras.

1. Early Computing Devices

  • ABACUS: Invented 5,000 years ago in China, it was the first mechanical device for calculation using beads on rods.
  • Napier's Bones: John Napier invented this in 1614 to perform multiplication and division using numbered rods.
  • Pascaline: Created by Blaise Pascal in 1642, it was the first mechanical calculator that used gears and wheels.
  • Analytical Engine: Designed by Charles Babbage (Father of Computers) in 1837, it was the first general-purpose programmable computer design.

2. Modern Computing Devices

  • ENIAC: The first general-purpose electronic computer built in 1946. It was huge and used vacuum tubes.
  • Personal Computers (PC): Developed in the 1970s and 80s, making computers accessible to individuals.
  • Smartphones & Tablets: Today's most common devices that offer high-speed processing, touch screens, and mobility.
  • Wearable Devices: Modern gadgets like Smartwatches that track health and provide notifications.

ii. Explain different classifications of computers.

Computers are classified based on their size, speed, and processing power:

1. Supercomputers

These are the most powerful and expensive computers in the world. They can perform trillions of calculations per second. They are used for complex tasks like weather forecasting, nuclear research, and space exploration.

2. Mainframe Computers

Large, powerful systems used by big organizations like banks and airlines. They can handle thousands of users simultaneously and process massive amounts of data reliably.

3. Minicomputers

Mid-sized computers that are smaller than mainframes but more powerful than microcomputers. They are used in small businesses for specialized tasks.

4. Microcomputers

Also known as Personal Computers (PC). These are designed for individual use. Examples include Desktops, Laptops, Tablets, and Smartphones. They are used in homes, schools, and offices.

iii. What is software? Explain its two main types with examples.

Software is a collection of programs and data that tells the computer hardware exactly what to do and how to perform a task.

1. System Software

It acts as an interface between the user, the application software, and the hardware. It controls the internal operations of the computer.

  • Operating System: e.g., Windows 10, Linux, Android.
  • Device Drivers: Software for printers, cameras, etc.
  • Utility Software: Antivirus, Disk Cleanup.

2. Application Software

It is designed to help the user perform specific tasks on the computer.

  • General Purpose: Word processors (MS Word), Browsers (Chrome).
  • Special Purpose: Banking software, Hospital management systems.

iv. Explain different types of general purpose application software.

General purpose software is developed for the general public to perform various tasks:

  • Word Processing Software: Used to create, edit, and format text documents like letters and reports (Example: MS Word).
  • Spreadsheet Software: Used for calculations, data analysis, and creating charts using rows and columns (Example: MS Excel).
  • Presentation Software: Used to create visual slides for teaching or business meetings (Example: MS PowerPoint).
  • Database Management Systems (DBMS): Used to store and manage large amounts of organized data (Example: MS Access).
  • Multimedia Software: Used to play or edit audio and video files (Example: VLC Media Player).

v. Write short note on any five input devices.

  1. Keyboard: The most common input device used to enter text, numbers, and commands into the computer.
  2. Mouse: A pointing device that allows users to select objects and navigate the screen by clicking.
  3. Scanner: Converts physical documents or photos into digital format that can be stored on a computer.
  4. Microphone: Used to record sound or give voice commands to the computer.
  5. Digital Camera: Captures images and videos digitally, which can then be transferred to a computer for editing.

vi. What is output? Briefly write about softcopy output devices.

Output is the final result produced by the computer after processing the input data. Output can be in two forms: Softcopy and Hardcopy.

Softcopy Output Devices

Softcopy is the electronic version of output that is displayed on a screen or heard through speakers. It is temporary and cannot be touched.

  • Monitors: The most common device to display text and graphics. Types include LCD, LED, and CRT.
  • Speakers: Used to produce sound output like music, alerts, or speech.
  • Projectors: Used to display computer output on a large screen or wall, commonly used in classrooms and meetings.

vii. What is the importance of magnetic cards/devices based systems? Explain different types.

Magnetic devices store data using magnetized particles on a strip. They are important because they provide a quick and automated way to identify people and process transactions.

Types of Magnetic Devices:

  • Magnetic Stripe Cards: Common in credit/debit cards and ID cards. The black strip on the back stores the user's account details.
  • Magnetic Card Readers: Devices used at shops (POS terminals) or building entrances to read the data from the card's stripe.
  • MICR (Magnetic Ink Character Recognition): Used by banks to read special magnetic ink printed on the bottom of cheques for fast processing.

viii. What are impact and non-impact printers? Explain any two types of printers in each category.

Printers are categorized based on how they print on paper.

1. Impact Printers

They print by striking an inked ribbon against the paper. They are noisy but cheap.

  • Dot-Matrix Printer: Uses pins to create characters out of dots.
  • Daisy Wheel Printer: Uses a wheel with characters to strike the ribbon.

2. Non-Impact Printers

They print without striking the paper, using laser, heat, or ink spray. They are quiet and produce high quality.

  • Laser Printer: Uses a laser beam and dry ink (toner) to print very fast.
  • Inkjet Printer: Sprays tiny drops of liquid ink onto the paper, excellent for photos.

ix. How a plotter is different from a printer? Explain different types of plotters.

A printer is designed for standard text and images on small paper, while a **Plotter** is a specialized device used to print high-quality large-scale vector graphics like maps, blueprints, and engineering designs.

Types of Plotters:

  • Flatbed Plotter: The paper is placed on a flat surface, and a pen moves over it to draw the design.
  • Drum Plotter: The paper moves over a rotating drum while the pen moves across to create very long drawings.
  • Inkjet Plotter: Uses inkjet technology instead of pens to create large colored banners and posters.
