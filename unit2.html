Q2. Give short answers to the following questions.

i. What is computer memory?
Computer memory is the electronic workspace of a computer used to store data and instructions either temporarily or permanently. It is made up of millions of small storage locations, each having a unique address. Memory is essential because the CPU needs a place to hold data that it is currently processing. It is broadly categorized into primary memory (like RAM) for immediate tasks and secondary memory (like Hard Disks) for long-term storage of files and software.

ii. What is the importance of cache memory in a computer?
Cache memory is a small, extremely fast type of volatile memory located very close to or inside the CPU. Its primary importance lies in its ability to bridge the speed gap between the fast processor and the relatively slow main memory (RAM). By storing frequently used data and instructions, cache allows the CPU to access them much quicker, which significantly improves the overall speed and performance of the system. Without cache, the processor would spend most of its time waiting for data from the RAM.

iii. Define bit, byte and memory word.
A Bit (Binary Digit) is the smallest unit of data in a computer, representing either a 0 or a 1. A Byte consists of 8 bits and is the standard unit used to represent a single character, like a letter or a number. A Memory Word is the amount of data that a CPU can process or move in a single operation. The length of a memory word (e.g., 32-bit or 64-bit) determines how much information the processor can handle at once, directly impacting the system's efficiency.

iv. Give some uses of secondary memory.
Secondary memory is used for permanent storage of data that is not currently being processed by the CPU. Its main uses include storing the Operating System (like Windows), installing various application softwares, and keeping user files such as documents, photos, and videos for a long time. It is also used to create backups of important data to prevent loss. Unlike primary memory, secondary memory is non-volatile, meaning it keeps the data safe even when the power is turned off.

v. What is the role of registers in computer?
Registers are high-speed storage locations located directly inside the CPU. Their role is to hold data, instructions, or memory addresses that are being processed immediately. They are the fastest form of memory in a computer system but have very small storage capacity. Registers help in performing arithmetic and logical operations by providing the CPU with instant access to the operands. Different types of registers, like the Accumulator or Program Counter, perform specific tasks to manage the execution of programs.

vi. Differentiate between DRAM and SRAM.
Dynamic RAM (DRAM) is the most common type of main memory; it is cheaper and has high storage density but requires constant electrical refreshing to keep data. In contrast, Static RAM (SRAM) is much faster and does not need to be refreshed, but it is more expensive and consumes more power. DRAM is typically used for a computer's main RAM, while SRAM is used for the CPU's cache memory because of its superior speed. SRAM holds data as long as power is supplied, whereas DRAM loses data if not refreshed thousands of times per second.

vii. Give few characteristics of secondary storage devices.
Secondary storage devices have several key characteristics: 1. Non-volatile: They retain data even without power. 2. High Capacity: They can store massive amounts of data (GBs and TBs). 3. Permanence: Data remains stored until the user deletes it. 4. Portability: Many devices like USBs or External Hard Drives can be moved between computers. 5. Slower Speed: They are significantly slower than primary memory because they often involve mechanical parts or optical lasers to read data.

Q3. Give detailed answers to the following questions.

i. Briefly write about the processor internal memory and its types.

Processor internal memory refers to the high-speed storage locations directly accessible by the CPU to ensure fast execution of instructions. It primarily consists of Registers and Cache memory.

1. Registers

Registers are the fastest storage units in the computer, located inside the CPU. They hold data currently being processed. Types include:

  • Accumulator: Stores the results of arithmetic and logic operations.
  • Program Counter (PC): Holds the address of the next instruction to be executed.
  • Instruction Register (IR): Holds the current instruction being decoded.

2. Cache Memory

Cache is a high-speed buffer between the CPU and RAM. It stores frequently used data to speed up processing. It has three levels:

  • L1 Cache: Built inside the CPU chip, fastest but smallest.
  • L2 Cache: Located on or near the CPU, slower than L1 but larger.
  • L3 Cache: Shared between multiple CPU cores, largest but slowest of the three.

ii. Explain RAM and ROM along with their types in detail.

Main memory is divided into RAM and ROM, each serving a distinct purpose in the computer system.

1. RAM (Random Access Memory)

RAM is volatile memory used for temporary storage of data during active processing. Its types include:

  • DRAM (Dynamic RAM): Needs constant refreshing; used as main memory.
  • SRAM (Static RAM): Faster, no refreshing needed; used for cache.
  • SDRAM: Synchronized with the system clock for better efficiency.

2. ROM (Read Only Memory)

ROM is non-volatile memory used to store permanent instructions like the BIOS. Its types include:

  • PROM: Can be programmed only once by the user.
  • EPROM: Can be erased using Ultraviolet light and then reprogrammed.
  • EEPROM: Can be erased and reprogrammed electrically, commonly used in modern BIOS chips.

iii. What is meant by secondary storage devices also explain the difference between Sequential and Random access.

Secondary storage devices are external or internal non-volatile memory used for long-term data storage. The way data is accessed on these devices defines their performance.

1. Sequential Access

In sequential access, data must be read in a specific order from beginning to end. If you want to access the 10th file, you must pass through the first 9. Example: Magnetic Tapes. It is slow but useful for backups.

2. Random Access (Direct Access)

In random access, the device can jump directly to any specific piece of data regardless of its location. This is much faster for daily tasks. Example: Hard Disks, SSDs, and RAM.

iv. Describe Magnetic tapes, Magnetic disks, and Optical disks with advantages/disadvantages.

(a) Magnetic Tapes

Used for sequential storage on a thin plastic ribbon coated with magnetic material.

  • Advantage: Very cheap and durable for long-term backups.
  • Disadvantage: Extremely slow data access due to sequential nature.

(b) Magnetic Disks (Hard Disk)

Uses spinning platters and read/write heads to store data magnetically.

  • Advantage: Large storage capacity and fast random access.
  • Disadvantage: Sensitive to physical shocks and magnets.

(c) Optical Disks (CD, DVD, Blu-Ray)

Uses laser technology to read and write data on a reflective surface.

  • Advantage: Very cheap and portable.
  • Disadvantage: Scratches easily and has slower speeds than hard disks.

v. Describe the advantages and disadvantages of Flash memory.

Flash memory is a type of EEPROM that is solid-state (no moving parts) and used in USB drives and SSDs.

Advantages:

  • Speed: Much faster than hard disks or optical disks.
  • Durability: No moving parts, so it is resistant to physical shocks.
  • Portability: Small size makes it ideal for mobile devices.
  • Power: Consumes very little electricity.

Disadvantages:

  • Cost: Generally more expensive per GB than hard disks.
  • Life Span: Has a limited number of write/erase cycles before it fails.
  • Capacity: While growing, it still trails hard disks in maximum available storage for the price.
